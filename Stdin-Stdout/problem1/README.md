# Java Stdin and Stdout I

## Problem Statement

In this challenge, read **three integers** from standard input and print each integer on a new line.

A portion of the code is already provided. Complete the program by reading the input using the `Scanner` class and printing the values in the same order.

## Input Format

- The input consists of **3 lines**.
- Each line contains a single integer.

## Output Format

Print each integer on a separate line.

## Sample Input

```text
42
100
125
```

## Sample Output

```text
42
100
125
```

## Java Concepts Used

- `Scanner` class
- `nextInt()`
- `System.out.println()`
- Standard Input and Output

## Solution

The program:
1. Creates a `Scanner` object.
2. Reads three integers using `nextInt()`.
3. Prints each integer on a new line.
4. Closes the scanner.
