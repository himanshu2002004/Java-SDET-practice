# Java Loops I

## Problem Statement

In this challenge, use a loop to print the multiplication table of a given integer.

Given an integer `N`, print its first **10 multiples**. Each multiple should be printed on a new line in the following format:

```text
N x i = result
```

where `i` ranges from **1** to **10**.

## Input Format

- A single integer `N`.

## Output Format

Print **10 lines** of output. Each line should contain the multiplication of `N` and `i` in the following format:

```text
N x i = result
```

where:

- `N` is the given integer.
- `i` ranges from **1** to **10**.
- `result = N × i`.

## Sample Input

```text
2
```

## Sample Output

```text
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
2 x 4 = 8
2 x 5 = 10
2 x 6 = 12
2 x 7 = 14
2 x 8 = 16
2 x 9 = 18
2 x 10 = 20
```

## Java Concepts Used

- `Scanner` class
- `for` loop (or `while` loop)
- Integer arithmetic
- String concatenation
- Console output using `System.out.println()`

## Learning Outcome

This challenge demonstrates how to:

- Read integer input using `Scanner`.
- Use loops to execute a block of code repeatedly.
- Perform multiplication inside a loop.
- Format output according to the required pattern.
